		Сопроводительная записка

	1. О программе
Данная программа представляет собой мою реализацию калькулятора Windows, которая написана для закрепления навыков. 
Основной функцианал был взят оттуда и старался как можно сильнее скопировать интерфейс и поведение программы.

	2. Цели
В первую очередь преследовалась цель  поднять навыки написания кода и программ,развязать логику и представление,
написание юнит тестов,документрирование написанного кода.

	3. Состав
Решение состоит их трех проектов -
a) Calc_logics			- Класс в котором содержится логика калькулятора.
б) Calc_Forms			- Форма WinForm - интерфейс и взаимодействие с логикой
в) Calc_Logic_Tests		- Нeсколько классов юнит тестов.

Calc_logics - Статический класс с логикой, содержит публичное свойство для предоставления данных Logics.Data
 и методы для ввода, редактирования и подсчета записанных в него чисел

Calc_Forms - Форма WinForm, нарисованная по образу калькулятора Windows и предлагающая схожее взаимодействие с пользователем путем нажатий на соответсвующие клавиши 
графического интерфейса.

Calc_Logic_Tests - Классы юнит тестов, тестирующих редактирование чисел(Logic_Input_Tests), арифметику(Logic_Arifmetic_Test), и доп. математику(Logic_adMath_Test)
покрыть код на 100% можно было и с меньшим числом тестов, но тесты проверяют и различные ситуации которые при постом покрытии не выявить

	4. Итог
В отсутсвие знающий людей с которыми можно было бы проконсультироваться, и написания кода наобум - т.е. без изучения специальной литературы о написании программ,
имея лишь базовые знания языка, считаю? что с поставленной задачей справился, пусть и не самым оптимальным способом.
В процессе использовались полученные знания, здравый смысл и сайты - stackoverflow.com и msdn.com
